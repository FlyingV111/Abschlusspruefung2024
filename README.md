    o ISO 2700x
    o BSI IT-Grundschutz
- Datenschutzgesetze – national und auf EU-Ebene, z.B. DSGVO, BDSG
    o Definition von personenbezogenen Daten
- Vertraulichkeit, Verfügbarkeit, Integrität, Authentizität, Verantwortlichkeit
- Phishing
- Hacking

    o Kooperativ
